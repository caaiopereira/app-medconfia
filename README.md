    ◼  Defina sua ideia:<br>
>- Pense que todo produto inovador precisa resolver um problema<br>

3 - Público Alvo:<br>

>- O que vê?<br>
>- O que fala e faz?<br>
>- Quais são as dores e suas necessidades?<br>
